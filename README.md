# 📈 QuantLab – Algorithmic Trading Platform

A professional algorithmic trading and backtesting platform built with **Python**, **Streamlit**, **VectorBT**, **Plotly**, and **Yahoo Finance**. QuantLab enables users to test multiple quantitative trading strategies, visualize performance, and analyze portfolio metrics through an interactive dashboard.

---

## 🚀 Features

- 📊 Interactive Streamlit Dashboard
- 📈 Real-time Historical Stock Data using Yahoo Finance
- 🇮🇳 Indian & 🇺🇸 US Stock Support
- 🔄 Multiple Algorithmic Trading Strategies
  - SMA Crossover
  - RSI Mean Reversion
  - MACD
  - Momentum
  - Bollinger Bands
  - Donchian Breakout
- 📉 Portfolio Backtesting using VectorBT
- 📊 Advanced Performance Metrics
- 📈 Interactive Plotly Visualizations
- 💾 SQLite Database Integration
- 📥 Trade History Export (CSV)
- 📚 Backtest History Tracking

## 📊 Performance Metrics

QuantLab evaluates strategies using:

- Total Return
- Buy & Hold Return
- Annual Return (CAGR)
- Annual Volatility
- Sharpe Ratio
- Sortino Ratio
- Calmar Ratio
- Maximum Drawdown
- Win Rate
- Profit Factor
- Total Trades
